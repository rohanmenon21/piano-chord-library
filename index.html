Help & Instructions

How to Use Piano Chord Library

This guide walks through every major feature in the app, from saving your first song to using setlists and performance mode live.

Getting Started

Piano Chord Library is a private online song organizer for chord sheets. Each user signs into their own account and only sees their own songs, setlists, and saved transpositions.

  1. Sign in or create an account from the login screen.
  2. Use New Song to start a blank song.
  3. Paste your lyrics and chord sheet into the song editor.
  4. Switch to Preview to see the formatted result.

Use this first when you want a simple home for your music library that stays synced to your account across devices.

Creating and Editing Songs

Songs are created from the left sidebar and edited in the main workspace. Draft songs appear at the top until they are saved.

  1. Click New Song.
  2. Enter the song title, artist, and original key.
  3. Paste the song content into the large text area.
  4. Click Save Song or let autosave store your changes.

This is best for adding your own custom arrangements, church charts, or personal lyric sheets.

Edit and Preview Tabs

The workspace has two tabs so writing and reading stay separate.

  1. Use Edit to change the raw song text and metadata.
  2. Use Preview to see chords highlighted above the lyrics.
  3. Selecting an existing saved song opens it in Preview.
  4. Creating a new song starts you in Edit.

Use Preview when you want a cleaner playing view without editing controls in the way.

Transposing and Saved Keys

Every song keeps both its original key and its currently saved transposed key. The app remembers the saved transpose for that song.

  1. Open a song in Preview or Performance Mode.
  2. Use - Semitone or + Semitone to shift the chords.
  3. Watch the key badges update to show the original key and saved key.
  4. Use Reset to go back to the original arrangement.

This is useful when a singer needs a different key but you still want to remember the chart’s original starting point.

Import From URL

The import tool tries to fetch a chord-and-lyrics page and pull the song text into the current draft automatically.

  1. Open the Edit tab.
  2. Paste a web page link into Import From URL.
  3. Click Import.
  4. Review the imported result and save the song.

Use this when you already found a chord chart online and want to bring it into your own library quickly. Some websites may still block imports.

Sorting and Search

The song list can be filtered and sorted to make a growing library easier to manage.

  1. Use the Sort by dropdown above the song list.
  2. Choose Most Recent Viewed, Song Name, or Artist Name.
  3. Type in the search box to filter the visible song list.
  4. Your sort preference is saved to your account automatically.

Use recent viewed during rehearsals, and alphabetical sorts when you want to clean up or browse the full library.

Deleting Songs and Undo

Deleting a song removes it right away, but you get a short undo window in case it was accidental.

  1. Select a saved song.
  2. Click Delete Song.
  3. Use the Undo popup if you want to restore it.
  4. If the undo window expires, the song stays deleted.

This is helpful when cleaning your library while still giving yourself a quick recovery option.

Chord Hover Preview and Voicings

In Preview, real chords are highlighted. Hovering them opens a piano view so you can see how to play the chord visually.

  1. Open a song in Preview.
  2. Hover over a highlighted chord name.
  3. Use the small left and right arrows in the popup to change voicings.
  4. Move away and come back to the same chord to see your last selected voicing again.

Use this when you want quick on-screen chord guidance without leaving the song chart.

Setlists

Setlists let you build ordered groups of saved songs for a service, performance, or practice session.

  1. Click New Setlist in the sidebar.
  2. Name the setlist and use Add Song to include songs from your library.
  3. Use Up and Down to reorder items.
  4. Click a song inside the setlist editor to open it in Preview.

Use setlists when you want a fixed song order for a rehearsal, worship set, or event.

Performance Mode

Performance Mode opens a cleaner full-screen reading view for live playing.

  1. Open a song in Preview.
  2. Click Performance Mode.
  3. If you opened it from a setlist song, use Previous and Next to move through the setlist.
  4. Click Exit to return to the main app.

Use this on stage or in rehearsal when you want large, distraction-free song text.

Auto-Scroll

Auto-scroll is available in both Preview and Performance Mode for longer songs that need hands-free scrolling.

  1. Set the Speed value between 0.1 and 1.0.
  2. Click Start to begin scrolling.
  3. Click the same button again to Pause, then Resume.
  4. The controls stay visible while the song scrolls.

This is useful when you are playing continuously and do not want to touch the trackpad or screen as often.

Account and Profile

Your account page manages the personal information tied to your private hosted library.

  1. Click Account in the top-right corner.
  2. Update your display name if needed.
  3. Review the email address attached to the account.
  4. Use Sign Out when you are done, especially on shared devices.

Use this to keep your profile accurate and make sure you are working in the correct personal library.
